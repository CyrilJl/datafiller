# Scripts

Utility scripts for maintainers. The three `*.py` scripts regenerate the static assets in `docs/_static/`
(benchmark results and example CSVs rendered by the documentation pages); `multivariate_benchmark.py` can
also be run standalone to compare imputer performance.

Run them all from the repository root with `scripts/run_scripts.sh` (or `scripts\run_scripts.bat` on Windows).
